# Notes

## English

### What it is

A one-line micro-journal that appears right after you complete a habit.

### Highlights

- Prompt on every completion
- Short reflections only — not a full diary
- Saved notes in plugin settings
- No network required

### Permissions

- `storage` — journal entries
- `ui` — reflection sheet

### How to use

Install from Settings → Plugins. Complete a habit; enter one short line when the sheet opens. Reread saved notes in plugin settings.

## Русский

### Что это

Микро-журнал в одну строку — появляется сразу после отметки привычки.

### Возможности

- Запрос после каждой отметки
- Только короткие фразы — не дневник
- Сохранённые заметки в настройках
- Без сети

### Разрешения

- `storage` — записи журнала
- `ui` — лист рефлексии

### Как пользоваться

Установите в Настройки → Плагины. Отметьте привычку и введите одну короткую фразу. Перечитайте заметки в настройках плагина.
