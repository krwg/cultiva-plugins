# Time

## English

### What it is

A live digital clock in the Cultiva header, with a sheet for extra time zones.

### Highlights

- Always-on header clock
- Multiple world time zones in one sheet
- Format and accent color in settings
- No network required

### Permissions

- `storage` — format and color
- `ui` — header widget and sheet

### How to use

Install from Settings → Plugins. The clock appears in the header. Tap it to open world times. Change format and color in plugin settings.

## Русский

### Что это

Живые цифровые часы в шапке Cultiva и лист с дополнительными часовыми поясами.

### Возможности

- Часы всегда в шапке
- Несколько мировых поясов в одном листе
- Формат и цвет в настройках
- Без сети

### Разрешения

- `storage` — формат и цвет
- `ui` — виджет шапки и лист

### Как пользоваться

Установите в Настройки → Плагины. Часы появятся в шапке. Нажмите, чтобы открыть мировое время. Формат и цвет — в настройках плагина.
