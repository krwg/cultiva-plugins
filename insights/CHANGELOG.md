## [1.0.0] — 2026-07-18

### Added
- Garden widget with local habit correlation patterns
- Co-occurrence: when habit X is done, habit Y is more often done
- Weekday when streaks break most often
- Best consecutive shared streak for a habit pair
- EN/RU strings; requires Cultiva **2.0.0**+
- No network — analysis stays on device
