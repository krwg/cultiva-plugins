# Correlations

## English

### What it is

On-device habit correlation patterns in the garden — co-occurrence, streak-break weekdays, and linked pair streaks from your Cultiva history.

### Highlights

- Habits that rise together
- Weekday when streaks most often break
- Longest shared run for a habit pair
- No network, no telemetry

### Permissions

- `habits.read` — completion history
- `storage` — widget state
- `ui` — garden widget

### How to use

Requires Cultiva 2.0.0+. Install from Settings → Plugins. Place the widget in the garden; it refreshes when habits complete. All math stays on your device.

## Русский

### Что это

Локальные закономерности между привычками в саду — совместное выполнение, дни срыва серий и связанные серии пар из вашей истории Cultiva.

### Возможности

- Привычки, которые растут вместе
- День недели, когда серии чаще обрываются
- Самый длинный общий пробег пары
- Без сети и телеметрии

### Разрешения

- `habits.read` — история отметок
- `storage` — состояние виджета
- `ui` — виджет в саду

### Как пользоваться

Нужен Cultiva 2.0.0+. Установите в Настройки → Плагины. Виджет в саду обновляется после отметок. Все расчёты — на устройстве.
