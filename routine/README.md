# Routine

## English

### What it is

Morning and evening checklists in the garden, matched to habits you name in settings.

### Highlights

- Separate morning and evening lists
- Habits matched by name
- Garden checklist UI
- Uses your Cultiva habit list

### Permissions

- `habits.read` — habit names and status
- `storage` — morning/evening lists
- `ui` — garden widget

### How to use

Requires Cultiva 2.0.0+. Install from Settings → Plugins. Enter comma-separated habit names for morning and evening. Open the garden checklist to walk through the list.

## Русский

### Что это

Утренний и вечерний чеклист в саду — по именам привычек из настроек.

### Возможности

- Отдельные списки утра и вечера
- Сопоставление привычек по имени
- Чеклист в саду
- Берёт ваши привычки Cultiva

### Разрешения

- `habits.read` — имена и статус привычек
- `storage` — списки утра/вечера
- `ui` — виджет в саду

### Как пользоваться

Нужен Cultiva 2.0.0+. Установите в Настройки → Плагины. Укажите привычки через запятую для утра и вечера. Пройдите список в саду.
