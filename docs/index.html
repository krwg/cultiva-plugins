How it works

Install in three steps

No npm, no store account. Cultiva desktop pulls files directly from this registry.

1

Get Cultiva

Install the desktop app for Windows, macOS, or Linux from GitHub Releases.

2

Open Plugins

Settings → Plugins → Browse. The app fetches registry.json and lists the catalog.

3

Install & grow

Tap Install. Files are verified, sandboxed, and wired into your header or garden.

Security

Sandboxed by design

Plugins never get raw Node or DOM access. Permissions are declared and enforced.

sha256 integrity

Every file in registry.json has a hash. Mismatch aborts install before code runs.

Opaque iframe

Plugin JavaScript runs in an isolated sandbox; UI is bridged via postMessage RPC.

Permission gates

network, storage, and ui must be listed in manifest.json and are checked at runtime.

Local habits

Cultiva habit data stays on disk. Plugins only see what you grant via the sandbox API.

Building a plugin? Read the author guide, add your folder, run the sha256 script, and open a PR.
