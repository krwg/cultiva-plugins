How to install.

Step 1

Get Cultiva

Install the desktop app from GitHub Releases — Windows, macOS, or Linux.

Step 2

Open Plugins

Settings → Plugins → Browse. Cultiva fetches registry.json automatically.

Step 3

Install

Tap Install. Files are sha256-verified, sandboxed, and wired into your UI.
