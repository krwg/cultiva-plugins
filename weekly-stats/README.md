# Summary

## English

### What it is

A garden widget with your last 7 days of habit completion — bars and a weekly rate from Cultiva 2.0 history.

### Highlights

- Seven-day completion chart
- Weekly completion percentage
- Updates when habits change
- Data stays on your device

### Permissions

- `habits.read` — completion history
- `storage` — widget state
- `ui` — garden widget

### How to use

Requires Cultiva 2.0.0+. Install from Settings → Plugins. Place the widget in the garden; it refreshes as you check in habits.

## Русский

### Что это

Виджет в саду с выполнением привычек за последние 7 дней — столбцы и недельный процент из истории Cultiva 2.0.

### Возможности

- График за семь дней
- Процент выполнения за неделю
- Обновление при изменении привычек
- Данные остаются на устройстве

### Разрешения

- `habits.read` — история отметок
- `storage` — состояние виджета
- `ui` — виджет в саду

### Как пользоваться

Нужен Cultiva 2.0.0+. Установите в Настройки → Плагины. Виджет в саду обновляется по мере отметок.
