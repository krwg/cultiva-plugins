# Reminders

## English

### What it is

A soft in-app nudge once a day after your chosen hour if planned habits are still open.

### Highlights

- Configurable nudge hour
- Fires only when habits remain incomplete
- Quiet signal — not spam
- No network required

### Permissions

- `habits.read` — open habits
- `storage` — nudge timing state
- `ui` — in-app signal
- `settings.read` — app preferences used for timing

### How to use

Requires Cultiva 2.0.0+. Install from Settings → Plugins. Set the nudge hour in plugin settings. After that hour, incomplete habits can trigger a gentle reminder.

## Русский

### Что это

Мягкое напоминание в приложении раз в день после выбранного часа, если запланированные привычки ещё не выполнены.

### Возможности

- Настраиваемый час напоминания
- Срабатывает только при незавершённых привычках
- Тихий сигнал — без спама
- Без сети

### Разрешения

- `habits.read` — открытые привычки
- `storage` — состояние напоминания
- `ui` — сигнал в приложении
- `settings.read` — настройки приложения для времени

### Как пользоваться

Нужен Cultiva 2.0.0+. Установите в Настройки → Плагины. Задайте час в настройках. После него незавершённые привычки могут вызвать мягкое напоминание.
