# Timer

## English

### What it is

Classic pomodoro focus cycles in the Cultiva header — work and break without leaving the garden.

### Highlights

- Default 25/5 intervals (configurable)
- Countdown in the header
- Gentle chime when a phase ends
- No network required

### Permissions

- `storage` — work/break lengths
- `ui` — header widget and sheet

### How to use

Install from Settings → Plugins. Start a session from the header chip. Adjust work and break minutes in plugin settings.

## Русский

### Что это

Классические циклы «помидора» в шапке Cultiva — работа и перерыв, не покидая сад.

### Возможности

- Интервалы 25/5 по умолчанию (настраиваются)
- Обратный отсчёт в шапке
- Мягкий сигнал в конце фазы
- Без сети

### Разрешения

- `storage` — длительности работы и перерыва
- `ui` — виджет шапки и лист

### Как пользоваться

Установите в Настройки → Плагины. Запустите сессию из чипа в шапке. Минуты работы и перерыва — в настройках плагина.
